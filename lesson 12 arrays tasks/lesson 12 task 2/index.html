Write a function which will:

  1. Accept array of number and minimal number
  2. Output (via console.log()) array containing only number bigger than minimal.

Example:

const filteredArray = filterByMinimalNumber([0, 1, 2], 1);
console.log(filteredArray); // Should output [1, 2]
const filteredArray = filterByMinimalNumber([10, 15, 23], 20);
console.log(filteredArray); // Should output [23]
